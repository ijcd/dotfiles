# jj-mirror Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship a `jj-mirror` tool that mechanically maintains a "prime" (PR-clean) bookmark chain as a derived view of a "source" (WIP) bookmark chain in the same jj repo, so jj-vine can push the prime chain without churning GitHub PRs across resyncs.

**Architecture:** Single bash script at `dot_local/bin/executable_jj-mirror` shelling out to `jj` for all VCS operations. State-free — each invocation re-derives the intended prime chain from the current jj graph and mutates only what's stale. Prefixes (`source-prefix`, `prime-prefix`) are read from jj config with defaults, never hardcoded in the code path. All atomicity comes from `jj op restore` — any failure rolls back to a captured pre-sync operation ID.

**Tech Stack:**
- Bash (needs associative arrays; not POSIX sh).
- `jj` 0.43+ (uses `duplicate --destination`, `bookmark set`, `op log`/`op restore`, revsets `trunk()`, `mine()`, `bookmarks()`, ancestors syntax).
- `shasum -a 256` (BSD stock on macOS; **not** `sha256sum`).
- `jj-vine` (optional — only needed for `jj-mirror push`).
- Test harness: plain bash `test_*.sh` scripts driving scratch jj repos under `$TMPDIR`.

## Global Constraints

- **No hardcoded prefix strings.** Every reference to source/prime bookmark names must go through `$SOURCE_PREFIX` / `$PRIME_PREFIX` shell variables, loaded once at startup from jj config with defaults `wip/` and `pr/`. The strings `wip` and `pr` must appear only in default-fallback expressions.
- **State-free.** No files written outside the working tree jj already owns. No `.jj/mirror/` directory. No cache. Everything re-derived per invocation.
- **Atomic sync.** Every `sync` invocation calls `jj op log --limit 1 -T 'id ++ "\n"'` before making changes; on any error, `jj op restore` back to that ID.
- **Match existing script style.** Follow `dot_local/bin/executable_jj-catch-up`: banner comment header (usage lines with two-space indent extractable by `awk`), `set -euo pipefail`, `usage()` function that awks the header, short flags for common options.
- **No `jq`, python, node, or other runtime dependencies.**
- **Deploy via chezmoi.** File goes at `dot_local/bin/executable_jj-mirror` in the repo (source path). Actual install via `chezmoi apply` → `~/.local/bin/jj-mirror` with executable bit set.
- **Failure ordering.** If prefix validation fails at startup, exit code 2 and print message to stderr. Runtime errors (conflict, missing jj) exit 1 after rolling back.

```bash
# jj-mirror — maintain a "prime" bookmark chain as a derived view of a
# "source" bookmark chain. Prime is what jj-vine pushes as PRs; source is
# where WIP happens. Non-source bookmarks (local scaffolding) are ignored.
#
# Usage:
#   jj-mirror                    alias for `sync`
#   jj-mirror sync               update prime chain to match source chain
#   jj-mirror status             show threads, source/prime pairs, staleness
#   jj-mirror push               sync, then `jj vine submit --tracked`
#   jj-mirror abandon <name>     delete both <source-prefix>name and <prime-prefix>name
#   jj-mirror --help             print this help
#

#   source-prefix   default "wip/"
#   prime-prefix    default "pr/"
#

set -euo pipefail

usage() { awk 'NR==1{next} !/^#/{exit} {sub(/^# ?/,"");print}' "$0"; }

case "${1:-sync}" in
  --help|-h) usage; exit 0 ;;
  sync|status|push|abandon) : ;;  # implemented in later tasks
  *) echo "jj-mirror: unknown command: $1" >&2; usage >&2; exit 2 ;;
esac
```

```bash
# list_bookmarks_with_prefix PREFIX
# Emits: <name>\t<short-commit-id> per line, filtered by PREFIX.
# Empty PREFIX matches every bookmark.
list_bookmarks_with_prefix() {
  local prefix=$1
  # Template: name TAB short commit_id NEWLINE. normal_target is the local commit.
  jj bookmark list -T 'name ++ "\t" ++ normal_target.commit_id().short() ++ "\n"' 2>/dev/null \
    | while IFS=$'\t' read -r name cid; do
        [[ -n "$name" && -n "$cid" ]] || continue
        if [[ -z "$prefix" || "$name" == "$prefix"* ]]; then
          printf '%s\t%s\n' "$name" "$cid"
        fi
      done
}

list_source_bookmarks() { list_bookmarks_with_prefix "$SOURCE_PREFIX"; }
list_prime_bookmarks()  { list_bookmarks_with_prefix "$PRIME_PREFIX"; }

strip_source_prefix() { printf '%s' "${1#"$SOURCE_PREFIX"}"; }
strip_prime_prefix()  { printf '%s' "${1#"$PRIME_PREFIX"}";  }
```

Add to `dot_local/bin/executable_jj-mirror`:

```bash
# list_source_threads
# Emits one line per thread: <leaf>\t<b1>|<b2>|...|<leaf>
# Algorithm:
#   1. Collect all source bookmarks with their commit ids.
#   2. For each pair, check ancestry via revset.
#   3. Leaves = bookmarks with no source-bookmark descendant.
#   4. For each leaf, walk `trunk()..leaf & <source bookmarks>` in ancestry order.
list_source_threads() {
  local -a names cids
  while IFS=$'\t' read -r name cid; do
    names+=("$name"); cids+=("$cid")
  done < <(list_source_bookmarks)

  local n=${#names[@]}
  (( n > 0 )) || return 0

  # For each candidate leaf L, ensure no other source bookmark has L as ancestor.
  local i j leaf other
  for (( i=0; i<n; i++ )); do
    leaf=${names[i]}
    local is_leaf=1
    for (( j=0; j<n; j++ )); do
      (( i == j )) && continue
      other=${names[j]}
      # Is 'other' a descendant of 'leaf'? If so, leaf isn't a leaf.
      if jj log --no-graph -r "descendants(\"$leaf\") & \"$other\"" -T 'commit_id ++ "\n"' 2>/dev/null \
         | grep -q .; then
        is_leaf=0; break
      fi
    done
    (( is_leaf )) || continue

    # Walk the ordered chain from trunk toward leaf.
    # jj log with a revset prints in reverse-chronological order (newest first);
    # we want oldest first, so reverse portably via awk. (macOS `tail -r` and
    # GNU `tac` both exist but neither is universal.)
    local chain
    chain=$(
      jj log --no-graph -r "trunk()..\"$leaf\" & bookmarks()" \
        -T 'bookmarks ++ "\n"' 2>/dev/null \
      | awk 'NF' \
      | awk '{a[NR]=$0} END{for(i=NR;i>=1;i--)print a[i]}'
    )

    # Each line may have multiple source-prefixed names comma-separated if a commit
    # carries several source bookmarks. Split, filter, join with |.
    local ordered=""
    while IFS= read -r line; do
      [[ -n "$line" ]] || continue
      IFS=',' read -ra parts <<< "$line"
      local part
      for part in "${parts[@]}"; do
        if [[ "$part" == "$SOURCE_PREFIX"* ]]; then
          ordered+="${ordered:+|}$part"
        fi
      done
    done <<< "$chain"

    printf '%s\t%s\n' "$leaf" "$ordered"
  done
}
```

```bash
# diff_hash REVSET
# Emit SHA-256 hex of the patch text of the revision's own diff.
# Uses -T "" to strip commit metadata; -p emits the diff.
diff_hash() {
  local rev=$1
  jj log --no-graph -r "$rev" -T '""' -p 2>/dev/null | shasum -a 256 | awk '{print $1}'
}
```

```bash
# capture_pre_op — echo the current jj op id, to feed jj op restore on failure.
capture_pre_op() {
  jj op log --limit 1 --no-graph -T 'id ++ "\n"' 2>/dev/null | head -n1
}

# find_prime_bookmark NAME — echo the commit id of <prime-prefix>NAME, or empty.
find_prime_bookmark() {
  local name=$1
  local full="${PRIME_PREFIX}${name}"
  jj log --no-graph -r "$full" -T 'commit_id.short() ++ "\n"' 2>/dev/null | head -n1
}

# sync_thread LEAF CHAIN_STR
# CHAIN_STR is "b1|b2|...|leaf" — pipe-separated source bookmark names in trunk-to-leaf order.
sync_thread() {
  local leaf=$1
  local chain_str=$2
  IFS='|' read -ra chain <<< "$chain_str"

  local prev
  prev=$(jj log --no-graph -r "trunk()" -T 'commit_id.short() ++ "\n"' | head -n1)

  local source_name suffix prime_full existing new_cid
  for source_name in "${chain[@]}"; do
    suffix=$(strip_source_prefix "$source_name")
    prime_full="${PRIME_PREFIX}${suffix}"
    existing=$(find_prime_bookmark "$suffix")

    if [[ -z "$existing" ]]; then
      # Cherry-pick source commit onto prev; capture new commit id.
      new_cid=$(jj duplicate "$source_name" --destination "$prev" \
                --template 'self.commit_id().short() ++ "\n"' 2>/dev/null | head -n1)
      # Older jj: no --template on duplicate; parse the "Duplicated <old> as <new>" line.
      if [[ -z "$new_cid" ]]; then
        new_cid=$(jj duplicate "$source_name" --destination "$prev" 2>&1 \
                  | awk '/Duplicated/ {print $NF; exit}')
      fi
      [[ -n "$new_cid" ]] || { echo "jj-mirror: duplicate produced no commit for $source_name" >&2; return 1; }
      jj bookmark set "$prime_full" -r "$new_cid"
      prev=$new_cid
    else
      prev=$existing   # placeholder — later tasks handle updates
    fi
  done
}

sync_main() {
  local pre_op
  pre_op=$(capture_pre_op)
  [[ -n "$pre_op" ]] || { echo "jj-mirror: could not capture pre-sync op id" >&2; exit 1; }

  local synced=0 line leaf chain
  while IFS=$'\t' read -r leaf chain; do
    [[ -n "$leaf" && -n "$chain" ]] || continue
    if ! sync_thread "$leaf" "$chain"; then
      echo "jj-mirror: thread $leaf failed; rolling back to op $pre_op" >&2
      jj op restore "$pre_op" >/dev/null 2>&1 || true
      exit 1
    fi
    synced=$((synced + 1))
  done < <(list_source_threads)

  printf 'jj-mirror: synced %d thread(s)\n' "$synced"
}
```

```bash
  local leaf=$1

  local source_name suffix prime_full existing existing_parent existing_hash source_hash new_cid
```

```bash
DRY_RUN=0
THREAD_FILTER=""

# parse_sync_flags — pull -n/--dry-run and -t/--thread out of "$@", leave the rest.
# Sets DRY_RUN and THREAD_FILTER as side effects. Echoes the remaining args
# for the caller to consume via `read -ra`.
parse_sync_flags() {
  local -a remaining=()
  while (( $# )); do
    case "$1" in
      -n|--dry-run) DRY_RUN=1; shift ;;
      -t|--thread)  THREAD_FILTER=${2:-}; shift 2 ;;
      --thread=*)   THREAD_FILTER=${1#--thread=}; shift ;;
      *) remaining+=("$1"); shift ;;
    esac
  done
  printf '%s\n' "${remaining[@]}"
}
```

Modify `sync_main` to honor these flags:

```bash
sync_main() {
  local pre_op
  pre_op=$(capture_pre_op)
  [[ -n "$pre_op" ]] || { echo "jj-mirror: could not capture pre-sync op id" >&2; exit 1; }

  local synced=0 skipped=0 leaf chain rc=0
  while IFS=$'\t' read -r leaf chain; do
    [[ -n "$leaf" && -n "$chain" ]] || continue
    if [[ -n "$THREAD_FILTER" ]]; then
      if [[ "$leaf" != "$THREAD_FILTER" && "|$chain|" != *"|$THREAD_FILTER|"* ]]; then
        skipped=$((skipped + 1)); continue
      fi
    fi

    if [[ $DRY_RUN -eq 1 ]]; then
      IFS='|' read -ra parts <<< "$chain"
      printf 'thread %s (dry-run)\n' "$leaf"
      local sname sfx existing
      for sname in "${parts[@]}"; do
        sfx=$(strip_source_prefix "$sname")
        existing=$(find_prime_bookmark "$sfx")
        if [[ -z "$existing" ]]; then
          printf '  would create %s%s from %s\n' "$PRIME_PREFIX" "$sfx" "$sname"
        else
          local hs hp
          hs=$(diff_hash "$sname"); hp=$(diff_hash "$existing")
          if [[ "$hs" == "$hp" ]]; then
            printf '  would keep %s%s (up-to-date)\n' "$PRIME_PREFIX" "$sfx"
          else
            printf '  would rebuild %s%s (diff changed)\n' "$PRIME_PREFIX" "$sfx"
          fi
        fi
      done
      continue
    fi

    if ! sync_thread "$leaf" "$chain"; then
      rc=$?
      echo "jj-mirror: rolling back to op $pre_op" >&2
      jj op restore "$pre_op" >/dev/null 2>&1 || true
      exit "$rc"
    fi
    synced=$((synced + 1))
  done < <(list_source_threads)

  if [[ $DRY_RUN -eq 1 ]]; then
    printf 'jj-mirror: dry-run — no changes made\n'
  else
    printf 'jj-mirror: synced %d thread(s)%s\n' "$synced" \
      "$([[ $skipped -gt 0 ]] && printf ' (%d skipped)' "$skipped")"
  fi
}
```
